# **Voice Chatbot Practice**
## 1. 프로젝트명
시작이 제일 무서워 미룬이

**제목 출처**
https://www.youtube.com/watch?v=yx1z7MbbKyM&list=RDyx1z7MbbKyM&start_radio=1

## 2. 상세 설명
기존의 챗봇(https://github.com/Yunu-Jeon/VOICE_Chatbot)이 사용자가 즉시 행동으로 옮기기에는 재미 요소가 부족하여 동기부여가 부족하다는 단점이 있다.
본 프로젝트는 사용자의 미루기 상태를 분석해 수행 가능한 최소 단위 퀘스트를 생성하고, 결과에 따라 동적으로 난이도를 조정하는 게임형 인터랙션을 제안한다.
  
## 3. 사용 방법
1. 우측 상단의 마이크 아이콘을 누르거나 채팅창을 통해 현재 미루고 있는 일을 말한다. ex) "내일 발표 자료 만들어야 하는데 너무 하기 싫어"
2. 제공된 로드맵을 확인하고, 해당 일을 바로 시작한다. (1분 동안 아무런 화면 조작이 없다면 백그라운드 타이머가 작동하여 팝업 알림을 띄워 주의를 준다.)
3. 1단계를 완료했다면 "다 했어" 또는 "완료"라고 말한다.
4. 로드맵 진행률(%)이 오르고 다음 퀘스트가 주어진다. (총 5단계)
5. 만약 로드맵이 마음에 안 들거나 잘못 인식되었다면 "아니, 다시 짜 줘"라고 말하거나 [🔄 로드맵 초기화] 버튼을 누른다.
