## 2. 상세 설명
기존의 챗봇(https://github.com/Yunu-Jeon/VOICE_Chatbot)이 사용자가 즉시 행동으로 옮기기에는 재미 요소가 부족하여 동기부여가 부족하다는 단점이 있다.
그 점을 반영하여 할 일을 시작하고 마무리하는 것이 어려운 사용자를 위해 현재 상태를 반영한 퀘스트 5단계를 생성하고, 
단계별 보상과 피드백을 주며 자신만의 정원을 길러나가는 게임형 챗봇/음성 인터랙션 시스템이다.
